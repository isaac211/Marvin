First install flask
pip install Flask

Then run start.sh

On your device, open a browser and visit 127.0.0.1:5000 to see the page
and verify that it works locally. 

You can also view the page from another device on the same network. Get
your IP address from ifconfig (linux) or ipconfig (windows) and visit
<IP_Address>:5000 to see the same page.

This has been tested with Firefox for Windows and iPhone.
